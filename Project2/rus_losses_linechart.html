I chose this dataset because I wanted to explore using multiple columns in a new graph format that I have not used before, and the dataset is updated daily on Kaggle, so there is room in the future to use it in Tableau and auto-update through API.
I chose a line graph as a clear choice when showing multiple values changing over time. I mainly used an SVG element in this visualization, but also included data loading, scaling on the axes, line generation, tooltips, color scaling, and finally a legend.
This project was full of challenges, choosing a dataset was difficult, and I wasn't certain if this subject type would be palatable enough for school work. I am not very good with Javascript and honestly, I really don't care for visualizations in it, and had to look up a framework for the line graph as well as syntax at every stage of the build. Just moving the legend around took quite a bit of effort, as it wanted to either be right in the middle of the graph, or when moved to the correct spot, would be cut off by the dimensions of the margin.
